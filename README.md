# 微信自动发送工具
本地 npm run start

打包 npm run dist

从生成的dist文件中，安装exe即可


# 使用方式
安装后，先编辑文本、或者图片，再点击【执行】，并把鼠标移动进入输入框，可以连续执行，并把已编辑的内容文、图片，发送给目标微信用户。
